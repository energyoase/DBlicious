# U4 — Multi-Step-Wizard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Eine wiederverwendbare Mehr-Schritt-Komponente, die Schritte deklarativ aus `wizard.json` lädt, Server-seitig State pro Wizard-Session hält, computed Step-Inputs vom Server liefert und am Ende auf einen U2-SavePreview oder einen U6-Run dispatched. Der `<FieldRenderer>` aus dem Editor wird wiederverwendet — keine doppelte Form-Implementation.

**Architecture:** Wire-Typen `WizardDefinition`, `WizardStep`, `WizardField`, `WizardComputed`, `WizardCompletion`, `WizardSessionView` in `shared::wizard`. Server: `WizardSession`-Store (DashMap + 1h-TTL, Pattern aus U2-Preview), Validator-Registry, Computer-Registry. Vier neue GraphQL-Mutations + zwei Queries. Client: vertikaler Stepper (links) + Step-Form (rechts), `<FieldRenderer>`-Reuse via Adapter, Routing `/wizards/:id`. Completion dispatched intern auf U2 oder U6 — beide sind eigenständige Plans und müssen zur Wizard-Implementation **nicht** schon gelandet sein (Stub-Pfad rendert nur "fertig" wenn weder U2 noch U6 verfügbar).

**Tech Stack:** Rust, `shared` (serde), `server` (axum + async-graphql + DashMap + tokio), `client` (Leptos + leptos_router 0.7), `cargo test --workspace`.

**Spec-Referenz:** [`docs/superpowers/specs/2026-05-20-u4-wizard-design.md`](../specs/2026-05-20-u4-wizard-design.md).

**Vorbedingung:** keine harten. U2 und U6 sind parallele Plans; Wizard-Completion-Dispatch ist additiv (Stubs falls noch nicht verfügbar).

**Frontend-design Entscheidungen** (für Stepper + Step-Form):
1. **Vertikaler Stepper links** (4-Step+-tauglich, kann Sub-Status anzeigen) — Spec dictiert Variante B.
2. **4px solid Akzent-Bar links neben aktivem Step** (statt Background-Highlight): rein Border-basierter Indikator, der bei jeder Theme-Variante konsistent funktioniert. Akzent: `#0066cc` (Standard) — nicht hartcodieren, via `DesignSystem::wizard_step_active` ziehen.
3. **Monospace-Step-Nummerierung** (`01` / `02` / `03`) — konsistent mit Render-Reference-Pattern aus U1, scan-fähig auch bei vielen Steps.
4. **Optional-Step-Affordance**: italic-Label + `(opt)`-Tag (Mono-Font) + 30% Opacity wenn Step-Sub-Label-Done-Hinweis fehlt — visuell zurückhaltend, klar identifizierbar.
5. **Disabled-Next mit `title`-Tooltip**: `i18n: wizard-next-disabled`-Key wird per `title`-Attribut auf den Button gelegt. Hover-Tooltip zeigt den konkreten Grund (z.B. "Spalten-Mapping unvollständig"). Wenn der Server eine `next_disabled_reason` liefert, die statt der i18n-Default.

```rust
use serde::{Deserialize, Serialize};

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardDefinition {
    pub id: String,
    pub title_key: String,
    pub steps: Vec<WizardStep>,
    pub completion: WizardCompletion,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardStep {
    pub id: String,
    pub title_key: String,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub description_key: Option<String>,
    pub fields: Vec<WizardField>,
    #[serde(default)]
    pub computed: Vec<WizardComputed>,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub validator_id: Option<String>,
    #[serde(default)]
    pub optional: bool,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardField {
    pub key: String,
    pub label_key: String,
    pub field_type: crate::FieldType,
    #[serde(default)]
    pub required: bool,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub default: Option<serde_json::Value>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardComputed {
    pub key: String,
    pub handler_id: String,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub depends_on_step: Option<String>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(tag = "kind", rename_all = "camelCase")]
pub enum WizardCompletion {
    SavePreview { entity_type: String },
    RunAction { action_id: String },
    Custom { handler_id: String },
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardSessionView {
    pub session_id: String,
    pub current_step: u32,
    pub inputs: serde_json::Map<String, serde_json::Value>,
    pub computed: serde_json::Map<String, serde_json::Value>,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub next_step: Option<WizardStep>,
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub next_disabled_reason: Option<String>,
}

#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
#[serde(rename_all = "camelCase")]
pub struct WizardCompletionResult {
    pub kind: String, // "savePreview" | "actionStarted" | "custom"
    pub payload: serde_json::Value,
}
```

### Validator + Computer Registry

```rust
use async_trait::async_trait;
use std::collections::HashMap;
use std::sync::{Arc, OnceLock, RwLock};

#[async_trait]
pub trait WizardValidator: Send + Sync + 'static {
    async fn validate(&self, session: &WizardSession, step_id: &str, fields: &serde_json::Value)
        -> Result<shared::ValidationResult, anyhow::Error>;
}

#[async_trait]
pub trait WizardComputer: Send + Sync + 'static {
    async fn compute(&self, session: &WizardSession, computed: &shared::WizardComputed)
        -> Result<serde_json::Value, anyhow::Error>;
}

static VALIDATORS: OnceLock<RwLock<HashMap<String, Arc<dyn WizardValidator>>>> = OnceLock::new();
static COMPUTERS: OnceLock<RwLock<HashMap<String, Arc<dyn WizardComputer>>>> = OnceLock::new();

pub fn validators() -> &'static RwLock<HashMap<String, Arc<dyn WizardValidator>>> {
    VALIDATORS.get_or_init(Default::default)
}
pub fn computers() -> &'static RwLock<HashMap<String, Arc<dyn WizardComputer>>> {
    COMPUTERS.get_or_init(Default::default)
}
pub fn register_validator(id: &str, v: Arc<dyn WizardValidator>) { validators().write().unwrap().insert(id.into(), v); }
pub fn register_computer(id: &str, c: Arc<dyn WizardComputer>) { computers().write().unwrap().insert(id.into(), c); }

// Built-in Validators:
pub struct NonEmpty;
#[async_trait]
impl WizardValidator for NonEmpty {
    async fn validate(&self, _s: &WizardSession, _step_id: &str, fields: &serde_json::Value)
        -> Result<shared::ValidationResult, anyhow::Error>
    {
        let mut result = shared::ValidationResult::default();
        if let Some(obj) = fields.as_object() {
            for (k, v) in obj {
                if v.is_null() || v == &serde_json::json!("") {
                    result.push_error(format!("Feld {} darf nicht leer sein", k));
                }
            }
        }
        Ok(result)
    }
}
```

### Completion-Dispatch

```rust
async fn complete_wizard(session: WizardSession, completion: &shared::WizardCompletion)
    -> async_graphql::Result<shared::WizardCompletionResult>
{
    match completion {
        shared::WizardCompletion::SavePreview { entity_type } => {
            // Falls U2 verfügbar: assembled input → previewCreateEntity intern aufrufen
            // V1-Stub: liefert Counts aus Session-Inputs
            let payload = serde_json::json!({
                "entityType": entity_type,
                "assembledInputs": session.inputs,
                "stub": true,
            });
            Ok(shared::WizardCompletionResult { kind: "savePreview".into(), payload })
        }
        shared::WizardCompletion::RunAction { action_id } => {
            // Falls U6 verfügbar: POST /actions/start intern
            let payload = serde_json::json!({
                "actionId": action_id,
                "inputs": session.inputs,
                "stub": true,
            });
            Ok(shared::WizardCompletionResult { kind: "actionStarted".into(), payload })
        }
        shared::WizardCompletion::Custom { handler_id } => {
            // Custom-Handler-Registry
            let payload = serde_json::json!({"handlerId": handler_id, "stub": true});
            Ok(shared::WizardCompletionResult { kind: "custom".into(), payload })
        }
    }
}
```

Sobald U2/U6 gelandet sind, ersetzen die Stubs in einer Folge-Task die echten Aufrufe.

```rust
#[derive(Clone)]
pub struct WizardSession {
    pub id: String,
    pub definition_id: String,
    pub current_step: usize,
    pub inputs: BTreeMap<String, serde_json::Value>,
    pub computed: BTreeMap<String, serde_json::Value>,
    pub created_at: Instant,
    pub expires_at: Instant,
    pub user_id: Option<String>,
}

#[derive(Clone, Default)]
pub struct WizardSessionStore {
    sessions: Arc<DashMap<String, WizardSession>>,
}

impl WizardSessionStore {
    pub fn new() -> Self { Self::default() }
    pub fn insert(&self, s: WizardSession) { self.sessions.insert(s.id.clone(), s); }
    pub fn get_clone(&self, id: &str) -> Option<WizardSession> {
        self.sessions.get(id).map(|e| e.clone())
    }
    pub fn update<F>(&self, id: &str, f: F) -> Option<WizardSession>
    where F: FnOnce(&mut WizardSession),
    {
        let mut entry = self.sessions.get_mut(id)?;
        f(&mut *entry);
        Some(entry.clone())
    }
    pub fn remove(&self, id: &str) -> Option<WizardSession> {
        self.sessions.remove(id).map(|(_, v)| v)
    }
    pub fn purge_expired(&self) {
        let now = Instant::now();
        self.sessions.retain(|_, s| s.expires_at > now);
    }
}

use std::sync::OnceLock;
static GLOBAL: OnceLock<WizardSessionStore> = OnceLock::new();
pub fn set_global(s: WizardSessionStore) { let _ = GLOBAL.set(s); }
pub fn global() -> &'static WizardSessionStore {
    GLOBAL.get_or_init(WizardSessionStore::new)
}
```

```rust
pub async fn run_validator(
    validator_id: &str, session: &WizardSession, step_id: &str, fields: &serde_json::Value,
) -> Result<shared::ValidationResult, anyhow::Error> {
    let map = validators().read().unwrap();
    match map.get(validator_id).cloned() {
        Some(v) => v.validate(session, step_id, fields).await,
        None => Ok(shared::ValidationResult::default()),
    }
}

pub async fn run_computer(
    session: &WizardSession, computed: &shared::WizardComputed,
) -> Result<serde_json::Value, anyhow::Error> {
    let map = computers().read().unwrap();
    match map.get(&computed.handler_id).cloned() {
        Some(c) => c.compute(session, computed).await,
        None => Ok(serde_json::Value::Null),
    }
}
```

```rust
    &self, ctx: &Context<'_>, definition_id: String,

        .ok_or_else(|| async_graphql::Error::new("unknown wizard"))?;
    let id = uuid::Uuid::new_v4().to_string();

        id: id.clone(), definition_id,
        current_step: 0, inputs: std::collections::BTreeMap::new(),
        computed: std::collections::BTreeMap::new(),
        created_at: now,
        expires_at: now + std::time::Duration::from_secs(3600),
        user_id: crate::auth::current_user_id(ctx).ok(),
    };

        session_id: id,
        current_step: 0,
        inputs: serde_json::Map::new(),
        computed: serde_json::Map::new(),

async fn submit_wizard_step(
    &self, _ctx: &Context<'_>,
    session_id: String, step_id: String, fields: async_graphql::Json<serde_json::Value>,
) -> async_graphql::Result<async_graphql::Json<shared::WizardSessionView>> {
    let session = crate::wizard::store::global().get_clone(&session_id)
        .ok_or_else(|| async_graphql::Error::new("session expired or unknown"))?;
    let def = crate::data::wizard_by_id(&session.definition_id)
        .ok_or_else(|| async_graphql::Error::new("definition not found"))?;
    let step = def.steps.iter().find(|s| s.id == step_id)
        .ok_or_else(|| async_graphql::Error::new("step not in definition"))?;

    // Validator
    let validation = if let Some(vid) = &step.validator_id {
        crate::wizard::handlers::run_validator(vid, &session, &step_id, &fields.0).await?
    } else {
        shared::ValidationResult::default()
    };
    if validation.has_errors() {
        return Err(async_graphql::Error::new("validation failed").extend_with(|_, e| {
            e.set("validation", serde_json::to_value(&validation).unwrap_or_default());
        }));
    }

    // Inputs persistieren
    let updated = crate::wizard::store::global().update(&session_id, |s| {
        if let Some(obj) = fields.0.as_object() {
            s.inputs.insert(step_id.clone(), serde_json::Value::Object(obj.clone()));
        }
        s.current_step = (s.current_step + 1).min(def.steps.len().saturating_sub(1));
    }).ok_or_else(|| async_graphql::Error::new("session lost"))?;

    // Compute für nächsten Step
    let next_index = updated.current_step;
    let next_step = def.steps.get(next_index).cloned();
    let mut computed = updated.computed.clone();
    if let Some(ns) = &next_step {
        for c in &ns.computed {
            let val = crate::wizard::handlers::run_computer(&updated, c).await?;
            computed.insert(c.key.clone(), val);
        }
        crate::wizard::store::global().update(&session_id, |s| s.computed = computed.clone());
    }

    Ok(async_graphql::Json(shared::WizardSessionView {
        session_id, current_step: updated.current_step as u32,
        inputs: btreemap_to_jsonmap(&updated.inputs),
        computed: btreemap_to_jsonmap(&computed),
        next_step, next_disabled_reason: None,
    }))
}

async fn complete_wizard(
    &self, _ctx: &Context<'_>, session_id: String,
) -> async_graphql::Result<async_graphql::Json<shared::WizardCompletionResult>> {
    let session = crate::wizard::store::global().remove(&session_id)
        .ok_or_else(|| async_graphql::Error::new("session expired or unknown"))?;
    let def = crate::data::wizard_by_id(&session.definition_id)
        .ok_or_else(|| async_graphql::Error::new("definition not found"))?;
    let result = complete_wizard_internal(session, &def.completion).await?;
    Ok(async_graphql::Json(result))
}

async fn cancel_wizard(
    &self, _ctx: &Context<'_>, session_id: String,
) -> async_graphql::Result<bool> {
    crate::wizard::store::global().remove(&session_id);
    Ok(true)
}

fn btreemap_to_jsonmap(b: &std::collections::BTreeMap<String, serde_json::Value>) -> serde_json::Map<String, serde_json::Value> {
    let mut m = serde_json::Map::new();
    for (k, v) in b { m.insert(k.clone(), v.clone()); }
    m
}

async fn complete_wizard_internal(
    session: crate::wizard::WizardSession,
    completion: &shared::WizardCompletion,
) -> async_graphql::Result<shared::WizardCompletionResult> {
    use shared::WizardCompletion as C;
    match completion {
        C::SavePreview { entity_type } => Ok(shared::WizardCompletionResult {
            kind: "savePreview".into(),
            payload: serde_json::json!({
                "entityType": entity_type,
                "assembledInputs": session.inputs,
                "stub": true,
            }),
        }),
        C::RunAction { action_id } => Ok(shared::WizardCompletionResult {
            kind: "actionStarted".into(),
            payload: serde_json::json!({
                "actionId": action_id,
                "inputs": session.inputs,
                "stub": true,
            }),
        }),
        C::Custom { handler_id } => Ok(shared::WizardCompletionResult {
            kind: "custom".into(),
            payload: serde_json::json!({
                "handlerId": handler_id,
                "inputs": session.inputs,
            }),
        }),
    }
}
```

```rust
use crate::graphql::wizards;
use crate::i18n::t;

#[component]
pub fn Wizard(definition_id: String) -> impl IntoView {
    let def_signal = LocalResource::new({
        let id = definition_id.clone();
        move || {
            let id = id.clone();
            async move { wizards::fetch_wizard(&id).await.ok().flatten() }
        }
    });

    let session = RwSignal::<Option<WizardSessionView>>::new(None);
    let (current_step_idx, set_current_step_idx) = create_signal(0usize);
    let (completion_result, set_completion) = create_signal::<Option<shared::WizardCompletionResult>>(None);

    // Beim ersten Mount: startWizard
    create_effect({
        let id = definition_id.clone();
        move |_| {
            let id = id.clone();
            wasm_bindgen_futures::spawn_local(async move {
                if let Ok(view) = wizards::start_wizard(&id).await {
                    session.set(Some(view));
                }
            });
        }
    });

    view! {
        <Suspense fallback=|| view! { <p>{ t("loading") }</p> }>
            {move || def_signal.read().as_ref().and_then(|opt| opt.clone()).map(|def| {
                let def_for_stepper = def.clone();
                let def_for_form = def.clone();
                view! {
                    <div style="display:grid;grid-template-columns:200px 1fr;gap:16px;">
                        <Stepper steps=def_for_stepper.steps.clone() current_index=current_step_idx/>
                        <main>
                            {move || {
                                let s = session.get();
                                let Some(view_data) = s else { return view! { <p>{ t("loading") }</p> }.into_any(); };
                                if let Some(result) = completion_result.get() {
                                    return view! {
                                        <div>
                                            <h2>{ t("wizard-done") }</h2>
                                            <pre>{serde_json::to_string_pretty(&result).unwrap_or_default()}</pre>
                                        </div>
                                    }.into_any();
                                }
                                let Some(step) = view_data.next_step.clone() else {
                                    // Letzter Step erreicht → completeWizard
                                    let sid = view_data.session_id.clone();
                                    let set_completion = set_completion.clone();
                                    wasm_bindgen_futures::spawn_local(async move {
                                        if let Ok(res) = wizards::complete(&sid).await {
                                            set_completion.set(Some(res));
                                        }
                                    });
                                    return view! { <p>{ t("wizard-finalizing") }</p> }.into_any();
                                };

                                let sid = view_data.session_id.clone();
                                let step_id = step.id.clone();
                                let inputs_for_initial = view_data.inputs.get(&step_id).and_then(|v| v.as_object().cloned()).unwrap_or_default();
                                let computed_for_step = view_data.computed.clone();

                                view! {
                                    <StepForm
                                        step=step
                                        initial=inputs_for_initial
                                        on_submit=Callback::new(move |fields: serde_json::Value| {
                                            let sid = sid.clone();
                                            let step_id = step_id.clone();
                                            let session = session.clone();
                                            let set_idx = set_current_step_idx.clone();
                                            wasm_bindgen_futures::spawn_local(async move {
                                                match wizards::submit_step(&sid, &step_id, fields).await {
                                                    Ok(view) => {
                                                        set_idx.set(view.current_step as usize);
                                                        session.set(Some(view));
                                                    }
                                                    Err(e) => {
                                                        web_sys::console::error_1(&format!("submit failed: {e:?}").into());
                                                    }
                                                }
                                            });
                                        })
                                        on_back=None
                                        computed=computed_for_step
                                    />
                                }.into_any()
                            }}
                        </main>
                    </div>
                }
            })}
        </Suspense>
    }
}
```
